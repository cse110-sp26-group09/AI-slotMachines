# AI Token Slot Machine (Vanilla Web)

Tiny slot machine app using **HTML + CSS + JavaScript** and a few platform APIs, themed around AI token economies.

## Run

Open `index.html` in a browser (double-click it).

## Features

- Token balance persisted via `localStorage`
- “Temperature” slider that changes symbol odds
- Adjustable bet (slider + quick chips + number input)
- Daily token grant
- Token shop power-ups (spend tokens)
- Sound toggle + volume slider (Web Audio)
- Optional haptics toggle (Vibration)
- Big-win overlay + fireworks (canvas) that scale with payout
