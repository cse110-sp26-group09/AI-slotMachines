# AI Token Slot Machine (Vanilla Web)

Open `index.html` in a browser and press **Spin** (or hit `Space`).

Features:
- Tokens in / tokens out gameplay (with a satirical “AI tokens” theme)
- Crypto RNG (`crypto.getRandomValues`) for symbol picks
- Web Audio SFX + volume controls, vibration on supported devices
- Shop + Settings (Turbo Spin, Auto‑Spin, Max Bet upgrade, FX intensity)
- Local persistence via `localStorage` (balance, debt, bet, log)
- Share results via Share Sheet (if available) or Clipboard fallback
