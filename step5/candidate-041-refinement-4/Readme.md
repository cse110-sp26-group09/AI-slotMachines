## Token Casino (Vanilla Slot Machine)

Static slot machine app built with plain HTML/CSS/JS.

### Run

Open `index.html` in a browser.

### Notes

- Token balance and upgrades persist via `localStorage`.
- Optional sound uses the Web Audio API (disabled by default) with a volume slider.
- Bet size is configurable (1×–20×) and scales both cost and payouts.
- Big wins trigger bigger effects (banner + fireworks) plus optional haptics.
- A token shop lets you spend TOK on upgrades (and questionable “privacy” features).
- “Sell my data” increases telemetry and spin cost (because of course it does).
