# TokenGacha 3000

A tiny, vanilla web slot machine that lightly roasts AI hype: you *spend tokens* to spin and *win tokens* when the "model" behaves.

## Run

Open `index.html` in a browser.

## Controls

- Click **Spin** or press `Space`
- Toggle auto-spin with **Auto-spin** or press `A`

## Notes

- State persists in `localStorage` (tokens, spin count, log).
- Uses platform APIs like Clipboard, Vibration (if available), and Web Audio (if allowed).
