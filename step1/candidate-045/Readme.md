# Token Tumbler (Vanilla Slot Machine)

Open `index.html` in a browser to play.

For the full “platform APIs” experience (Clipboard, Notifications, etc.), serve the folder locally:

```sh
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- `Space`: Spin
- `M`: Max Spin
- `A`: Auto ×5
