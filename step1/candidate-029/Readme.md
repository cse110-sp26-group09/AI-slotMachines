# Token Slot (Vanilla Web)

A tiny, offline-capable slot machine that pokes fun at AI token economics: you **spend tokens** to **maybe win tokens**, while the “model” narrates your choices.

## Run

Open `index.html` in a browser.

If you want a local server (recommended for Service Worker / install prompts), run:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.
