# AI Slots — Token Laundromat (Vanilla Web)

Open `index.html` in a browser and press **Spin**.

## What it does

- Costs tokens to spin; matching symbols pays out tokens (scaled by spin cost).
- Saves progress with `localStorage` (so your questionable decisions persist).
- Uses platform APIs: Clipboard (copy audit log), Web Audio (beeps), Vibration (mobile), and file download (export stats).

## Reset

Use the **Reset** button to wipe saved state.
