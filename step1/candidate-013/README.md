# Token Burner 3000 (Vanilla AI Slot Machine)

Open `index.html` in a browser (or serve this folder with any static server).

- Token balance + stats persisted in `localStorage`
- Optional platform API flourishes (Web Audio, Vibration, Clipboard/Share when available)

Files:

- `styles.css`
- `app.js`
