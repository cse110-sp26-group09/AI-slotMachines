AI Slots (Vanilla)

Open `index.html` in a browser.

What it is
- A parody slot machine about AI: you spend fictional tokens to spin and win fictional tokens back.
- Uses only HTML/CSS/JavaScript and browser platform APIs (localStorage, Web Animations, Web Audio, Vibration, Share/Clipboard).

Notes
- Tokens have no value. This is just a toy app.
- If you enable Reduced Motion in your OS/browser, spins will snap instantly.
