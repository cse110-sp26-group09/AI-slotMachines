# AI Token Slot Machine (Vanilla Web)

A tiny, fully client-side slot machine built with plain HTML/CSS/JavaScript. It roasts AI “token economics”: you win tokens, spend tokens, and sometimes get invoiced.

## Run

- Open `index.html` in a browser.

## Controls

- **Spin**: costs 5 tokens
- **Claim daily tokens**: +25 tokens (once per local day)
- **Copy brag**: copies your last result to clipboard (falls back to a manual copy prompt if clipboard isn’t available)
- **Reset save**: clears tokens/stats (stored in `localStorage`)
