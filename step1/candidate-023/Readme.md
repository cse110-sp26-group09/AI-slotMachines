AI Slot Machine (Vanilla Web)

`TokenBandit.ai` is a satire slot machine about AI: you win tokens, spend tokens, get rate-limited, and occasionally get billed for “overhead”.

**Run**

- Quick: open `index.html` directly.
- Best (enables service worker + more platform APIs): `python3 -m http.server 8000` then open `http://localhost:8000`.

No dependencies. No build step.
