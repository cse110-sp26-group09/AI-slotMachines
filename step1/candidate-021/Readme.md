# AI Token Slot Machine (Vanilla Web)

Open `ai-token-slot/index.html` in a browser and press **Spin** (or hit `Space`).

Notes:
- Progress saves to `localStorage` (balance, debt, bet, log, toggles).
- For PWA install + offline caching, serve the folder (e.g. `python3 -m http.server`) so it runs on `http://localhost`.
