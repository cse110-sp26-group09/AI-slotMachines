# AI Token Slot Machine (Vanilla Web)

Open `index.html` in a browser and press **Spin** (or hit `Space`).

Features
- Tokens-in / tokens-out gameplay: each spin charges your balance, wins pay tokens back, and an “API bill” grows anyway.
- AI satire knobs: `Temperature` changes chaos, `Compliance tax` reduces winnings and adds fees.
- Platform APIs: `crypto.getRandomValues` (RNG), `localStorage` (persistence), Web Audio + `navigator.vibrate` (optional), Share Sheet / Clipboard, `<dialog>` (with `confirm()` fallback).
