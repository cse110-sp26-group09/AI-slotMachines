# Token Grinder — AI Slot Machine (Vanilla Web)

Open `index.html` in a browser and press **Spin** (or `S`).

## Features

- Token economy: spend tokens to spin, win/lose tokens on outcomes
- AI-themed symbols + sarcastic “system messages”
- Auto-spin toggle
- Bet cycling (5/10/20/50/100)
- Web platform APIs: `crypto.getRandomValues`, Web Audio API, vibration (if supported), optional service worker + manifest
- Persists balance + stats via `localStorage`
