# TokenGobbler 3000 (AI Slot Machine)

Vanilla **HTML + CSS + JavaScript** slot machine that roasts AI while you spend and win **tokens**.

## Run

- Open `index.html` in a browser.
- Controls: **Space** = spin, **A** = auto-spin.

## Notes

- Uses platform APIs: `localStorage` (save tokens/stats), Web Audio API (beeps), `navigator.vibrate` (mobile haptics), Web Share API (optional), and a Service Worker for offline use (only on `http(s)`).
