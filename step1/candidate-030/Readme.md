# TokenSpin™ — The AI Slot Machine

A tiny, no-build, vanilla HTML/CSS/JS slot machine that makes fun of AI token economics.

## Run

Open `index.html` in a browser.

If your browser blocks some APIs on `file://`, run a local server:

```sh
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Controls

- Click **SPIN** (or press `Space` / `Enter`)
- Toggle auto-spin with the button (or press `A`)
- Toggle sound with the button (or press `M`)
- Share/copy your last “receipt” with the button (or press `S`)
- Toggle fullscreen with the button (or press `F`)
