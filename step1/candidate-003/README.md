# Token Casino — AI Slot Machine (Vanilla)

Open `ai-slot-machine/index.html` in a browser.

Notes:
- Spin costs tokens (based on bet + temperature).
- Wins pay out multiples of your bet; special combos do special nonsense.
- Uses platform APIs: `localStorage` (save), `crypto.getRandomValues` (random), WebAudio (beeps), `navigator.share`/Clipboard (share), `navigator.vibrate` (haptics).
